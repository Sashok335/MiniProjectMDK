# Отчёт о разработке

## Этапы

1. **Инициация** — выбрана тема "Угадай число", определены user stories, создан MVP.
2. **Проектирование** — описана архитектура, структура БД, схема взаимодействия.
3. **Разработка (TDD)** — написаны тесты, затем реализован код: database.py → game.py → main.py.
4. **Тестирование** — пройдены автоматические тесты (21 тест, покрытие 80%+).

## Ответы на вопросы

**Что было самым сложным в тестировании?**
- Тестирование игрового цикла с вводом/выводом. Пришлось продумать инъекцию input/output функций.

**Как изменилось бы приложение, если бы вы сразу знали обо всех багах?**
- Сразу была бы заложена обработка некорректного ввода и подсчёт только валидных попыток.

**Что удалось:**
